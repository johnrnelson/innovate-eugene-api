Some curl Examples

Use this to get data from the live server. Notice that the actuall HTTP verb is not important.

curl --request GET --data '{ "service": "data", "view": "AllAssets" }' https://api.innovateeugene.com/api
curl --request POST --data '{ "service": "data", "view": "AllAssets" }' https://api.innovateeugene.com/api
curl --request POST --data '{ "service": "data", "view": "AllAssets" }' https://api.innovateeugene.com/api

Use this to get data from the your own debug server. 

curl --request GET --data '{ "service": "data", "view": "AllAssets" }' http://localhost:9118/api
